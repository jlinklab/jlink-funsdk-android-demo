### 用户注册（不绑定手机号和邮箱）
- 接口说明

```
/**
 * 注册账号
 *
 * @param userName 用户名
 * @param password 密码
 * @param ls       回调
 */
void register(String userName, String password, BaseAccountManager.OnAccountManagerListener ls);

```

- 参数说明

| 参数 | 类型| 描述|取值范围|
| - | - | - |-|
| userName|String| 用户名| 1、4-32位, 由中文/字母/数字组成, 但不能是纯数字 <br> 2、可通过<a href="#docs-hash-8">checkUserName</a>接口查询用户名是否存在|
| password|String| 密码| 8-64个字符|
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|

- 示例代码

```java
XMAccountManager.getInstance().register(userName, passWord, new BaseAccountManager.OnAccountManagerListener() {
    @Override
    public void onSuccess(int msgId) {
        //成功回调
    }

    @Override
    public void onFailed(int msgId, int errorId) {
      //失败回调，errorId错误码
    }

    @Override
    public void onFunSDKResult(Message message, MsgContent msgContent) {

    }
});

```

### 手机号注册
#### 1.发送手机号注册验证码
- 接口说明

```java
/**
 * 发送手机号注册验证码

 * @param userName 用户名
 * @param phoneNo  手机号
 * @param ls       回调
 * @return 接口是否成功，true 成功 false 失败
 */
boolean sendPhoneCodeForRegister(String userName, String phoneNo, BaseAccountManager.OnAccountManagerListener ls);

```
- 参数说明

| 参数 | 类型| 描述|取值范围|

| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|

- 示例代码
