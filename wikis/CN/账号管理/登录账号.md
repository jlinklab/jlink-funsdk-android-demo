## 接口总览
| 接口类 | 接口名称| 接口描述 | 备注 |
| - | - | - | - |
| LocalAccountManager| login| 临时登录|本地登录，设备信息保存到手机本地 |
| XMAccountManager| login| 账号登录| |
| XMAccountManager| loginByUnionId| 第三方授权登录|微信授权登录、Facebook授权登录等 |
| XMAccountManager| loginByCode| 通过验证码方式登录|手机号验证码或者邮箱验证码 |
## 临时登录
- 接口说明

```
/**
 * @param dbPath 将设备信息存到手机本地的路径
 * @param ls     回调
 * @return 调用接口是否成功
 */
boolean login(String dbPath, OnAccountManagerListener ls);
```
- 参数说明

| 参数 | 类型| 描述|备注|
| - | - | - |-|
| dbPath |String| 将设备信息存到手机本地的路径| |
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|

- 示例代码

```
String localPath = Environment.getExternalStorageDirectory();//手机本地路径
LocalAccountManager.getInstance().login(localPath, new BaseAccountManager.OnAccountManagerListener() {
            @Override
            public void onSuccess(int msgId) {
                 //成功回调
            }

            @Override
            public void onFailed(int msgId, int errorId) {
                //失败回调，errorId错误码
            }

            @Override
            public void onFunSDKResult(Message msg, MsgContent ex) {

            }
        })
```

## 第三方授权登录
- 接口说明

```
/**
 * 第三方授权登录
 *
 * @param unionId   授权ID
 * @param name      非必填 默认传""
 * @param loginType 登录类型 {@link com.manager.db.Define#LOGIN_BY_WECHAT 微信登录}
 *                  {@link com.manager.db.Define#LOGIN_BY_FACEBOOK Facebook登录}
 *                  {@link com.manager.db.Define#LOGIN_BY_GOOGLE Google登录}
 *                  {@link com.manager.db.Define#LOGIN_BY_CODE 验证码方式登录}
 *                  {@link com.manager.db.Define#LOGIN_BY_LINE Line登录}
 *                  {@link com.manager.db.Define#LOGIN_BY_PHONE_KEY 手机一键登录}
 * @param ls        回调
 * @return
 */
boolean loginByUnionId(String unionId, String name, int loginType, BaseAccountManager.OnAccountManagerListener ls);
```
- 参数说明

| 参数 | 类型| 描述|备注|
| - | - | - |-|
| unionId   |String| 授权ID| 微信/Facebook等授权返回的授权ID|
| name|String| 非必填 默认传""| |
| loginType |int| 登录类型| LOGIN_BY_WECHAT：微信登录 <br> LOGIN_BY_FACEBOOK Facebook：登录 <br>LOGIN_BY_GOOGLE Google：登录<br>LOGIN_BY_CODE：验证码方式登录<br>LOGIN_BY_LINE Line：登录<br>LOGIN_BY_PHONE_KEY：手机一键登录 |
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|

- 示例代码

```
XMAccountManager.getInstance().loginByUnionId("XXXXX", "", LOGIN_BY_WECHAT, new BaseAccountManager.OnAccountManagerListener() {
            @Override
            public void onSuccess(int msgId) {
                //成功回调
            }

            @Override
            public void onFailed(int msgId, int errorId) {
                //失败回调，errorId错误码
            }

            @Override
            public void onFunSDKResult(Message msg, MsgContent ex) {

            }
        })
```
