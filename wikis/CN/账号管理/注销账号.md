### 1.第一次调用删除接口（如果没有绑定过手机号或者邮箱，就会直接删除成功，如果绑定了会发送验证码到手机或者邮箱，然后要再次调用）
- 接口说明

```
/**
 * 删除账号
 *
 * @param code 验证码
 * @param ls   回调
 * @return
 */
boolean deleteXMAccount(String code, BaseAccountManager.OnAccountManagerListener ls);
```
- 参数说明

| 参数 | 类型| 描述|备注|
| - | - | - |-|
| code |String| 注销码| 第一次不需要输入|
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|
- 示例代码

```
XMAccountManager.getInstance().deleteXMAccount("", new BaseAccountManager.OnAccountManagerListener() {
             @Override
              public void onSuccess(int msgId) {
                  //成功回调
              }
          
              @Override
              public void onFailed(int msgId, int errorId) {
                //失败回调，errorId错误码
              }
          
              @Override
              public void onFunSDKResult(Message message, MsgContent msgContent) {
          
              }
        })
```
