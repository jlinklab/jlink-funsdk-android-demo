
## 获取设备列表
### 1.账号登录后获取设备列表（获取设备列表之前要确保已经登录账号）
- 示例代码

```
XMAccountManager.getInstance().getDevList();//获取设备列表 
XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo("devId");//根据设备序列号获取设备信息
```

### 2.临时登录后获取设备列表（获取设备列表之前确保已经临时登录）

- 示例代码

```
LocalAccountManager.getInstance().getDevList();//获取设备列表 
XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo("devId");//根据设备序列号获取设备信息
```

注：[登录账号](https://developer.jftech.com/docs/?menusId=ab9a6dddd50c46a6af8a913b472ed015&siderid=58b147ebcd3147c59ed7f57798c7b660&lang=zh)



### 获取设备状态
> 登录设备之前必须要先获取设备状态

#### 1.获取单个设备状态
- 接口说明

```java
/**
 * 从服务器更新设备状态
 *
 * @param ls       回调
 * @param devId    设备序列号
 */
void updateDevStateFromServer(BaseAccountManager.OnDevStateListener ls, String devId);

```
- 参数说明

| 参数 | 类型| 描述|备注|
| - | - | - |-|
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|
| devId    | String|设备序列号| |

- 示例代码

```
账号登录后获取设备状态
XMAccountManager.getInstance().updateDevStateFromServer(new BaseAccountManager.OnDevStateListener() {
    @Override
    public void onUpdateDevState(String devId) {
        //单个设备状态回调
        XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo(devId);
        //获取设备状态
        xmDevInfo.getDevState()
    }

    @Override
    public void onUpdateCompleted() {
        //所有设备状态获取结束回调
    }
},devId);

临时登录后获取设备状态
LocalAccountManager.getInstance().updateDevStateFromServer(new BaseAccountManager.OnDevStateListener() {
    @Override
    public void onUpdateDevState(String devId) {
        //单个设备状态回调
        XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo(devId);
        //获取设备状态
        xmDevInfo.getDevState()
    }

    @Override
    public void onUpdateCompleted() {
        //所有设备状态获取结束回调
    }
},devId);
```

#### 2.批量获取设备状态
- 接口说明

```java
/**
 * 从服务器更新设备状态
 *
 * @param devIdList 设备列表
 * @param listener  回调
 */
void updateAllDevStateFromServer(List<String> devIdList, BaseAccountManager.OnDevStateListener listener);
```
- 参数说明

| 参数 | 类型| 描述|备注|
| - | - | - |-|
| devIdList| List<String>|设备列表| |
| ls| OnAccountManagerListener |回调接口| 返回成功和失败，包括失败的错误信息|


- 示例代码


### 获取设备主控状态（部分低功耗设备支持）
- 示例代码

```
//获取设备状态成功后，通过XMDevInfo获取主控状态
XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo(devId);
//获取设备状态
xmDevInfo.getDevMasterCtrlState()
```

### 唤醒设备主控（部分低功耗设备支持）
- 示例代码

```java
/**
 * 开锁并唤醒主控
 *
 * @param position
 * @param devId
 */
@Override
public void wakeUpAndSendCtrl(int position, String devId) {
    HashMap<String, Object> subMap = new HashMap<>();
    subMap.put("Cmd", "1");//0：默认值不唤醒主控，1：表示唤醒主控
    HashMap<String, Object> rootMap = new HashMap<>();
    rootMap.put("ExtInfo", subMap);
    DeviceManager.getInstance().wakeUpAndSendCtrl(devId, new Gson().toJson(rootMap), new DeviceManager.OnDevManagerListener() {
        @Override
        public void onSuccess(String devId, int operationType, Object result) {
            ToastUtils.showLong("发送成功");
        }

        @Override
        public void onFailed(String devId, int msgId, String jsonName, int errorId) {
            ToastUtils.showLong("发送失败");
        }
    });
}

```

##### XMDevInfo
| 字段名 | 说明 |
| ------ | ------ |
| devId | 设备序列号|
| devName| 设备名称|
| devUserName| 设备登录名|
| devPassword| 设备登录密码|
| devIp| 设备IP地址|
| pid| 产品类型，和devType差不多，新产品中会替代devType|
| mac| MAC地址|
| devPort| 设备端口号|
| devType| 设备类型|
| devState| 设备在线状态|
| cloudState| 云服务状态|
| cloudExpired| 云服务过期状态|
| systemInfoBean| 缓存的设备信息|
| systemInfoExBean| 缓存的设备扩展信息|
| systemFunctionBean| 缓存的设备能力集|
| sdbDevInfo| 设备在线状态|
| cloudState| 云服务状态|
| cloudExpired| 云服务过期状态|
| systemInfoBean| 兼容老的SDK中的设备信息|
| chipOEMId| 芯片OEMId|
| devMasterCtrlState| 主控状态：用于低功耗设备并且该设备的主控和WiFi单片机是分开的，如果主控和WiFi模块是一体的，那么该状态可以不需要判断，该状态会和devState保持一致性|

##### DevState 设备状态枚举
| 字段名 | 说明|
| ------ | ------ |
| OFF_LINE：0| 离线|
| ON_LINE：1| 在线|
| SLEEP：2| 休眠中（低功耗设备）|
| WAKE_UP：3| 唤醒中（低功耗设备）|
| WAKE：4| 已唤醒（低功耗设备）|
| SLEEP_UNWAKE：5| 睡眠中 不可唤醒（低功耗设备）|
| PREPARE_SLEEP：6| 准备睡眠（低功耗设备）|

##### CloudState 云服务状态枚举
| 字段名 | 说明|
| ------ | ------ |
| CLOUD_GET_STATE：0| 获取状态中|
| CLOUD_NOT_SUPPORT：1| 不支持|
| CLOUD_NOT_OPEND：2| 未开通|
| CLOUD_NORMAL：3| 已经开通,正常使用|
| CLOUD_EXPIRED：4| 已经过期|
