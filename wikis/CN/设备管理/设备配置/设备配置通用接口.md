### 1. 获取设备配置
- 接口说明

```
DevConfigInfo create(DeviceManager.OnDevManagerListener listener, String... field);
```
- 参数说明

| 参数  |类型     | 描述     | 取值范围|
| :--------- | -------- |------ |------ |
| listener  |OnDevManagerListener  | 回调     | 非空|
| field  |String...    | 配置名称   | 非空|

- 示例代码

```
DevConfigInfo devConfigInfo = DevConfigInfo.create(new DevConfigManager.OnDevConfigResultListener() {
    @Override
    public void onSuccess(String devId, int msgId, Object result) {
       
    }

    @Override
    public void onFailed(String devId, int msgId, String s1, int errorId) {
    
    }

    @Override
    public void onFunSDKResult(Message message, MsgContent msgContent) {
       
    }
});
------------------------------大多数配置----------------------------------------
devConfigInfo.setJsonName(jsonName);//设置配置JsonName:通过JsonConfig类中找到你想要获取配置的JsonName信息
devConfigInfo.setChnId(chnId);//通道ID，如果针对整个设备的配置传-1，针对通道就传对应的通道号，比如第一通道就传0，第二传1以此类推
devConfigManager.getDevConfig(devConfigInfo);//获取配置
devConfigManager.setDevConfig(devConfigInfo);//保存配置

------------------------------其他特殊命令--------------------------------------

devConfigInfo.setJsonName(jsonName);//设置配置JsonName:通过JsonConfig类中找到你想要获取配置的JsonName信息
devConfigInfo.setJsonData(jsonData);//发送的json数据
比如重启设备
JSONObject obj = new JSONObject();
obj.put("Action", "Reboot");
String jsonData = HandleConfigData.getSendData(JsonConfig.OPERATION_OPMACHINE, "0x01", obj)
devConfigInfo.setJsonData(jsonData);//发送的json数据

devConfigInfo.setCmdId(cmdId);//命令ID:通过EDEV_JSON_ID类找到你要设置的命令ID
devConfigInfo.setChnId(chnId);//通道ID，如果针对整个设备的配置传-1，针对通道就传对应的通道号，比如第一通道就传0，第二传1以此类推
devConfigManager.setDevCmd(devConfigInfo);
```

### 2. 保存配置

- 接口说明

```
void saveConfig(String jsonName, int chnId, String jsonData)
```
- 参数说明

| 参数     | 类型  | 描述     | 取值范围|
| :--------- | -------- |----- |----- |
| jsonName| String   | 配置名称   | 参考[JsonConfig]|
| chnId  |int     | 通道号 | 大于0|
| jsonData | String  | 配置内容| 非空|

- 示例代码

```
 /**
  *
  * @param jsonName  配置名称 如 “Camera.WhiteLight”
  * @param chnId  通道号 如 -1
  * @param jsonData 配置内容 如 “{"MoveTrigLight":{"Duration":60,"Level":3},"WorkMode":"Auto","WorkPeriod":{"EHour":6,"EMinute":0,"SHour":18,"SMinute":0}}”
  */
presenter.saveConfig(jsonName, spConfigChn.getSelectedValue(), jsonData);
```
