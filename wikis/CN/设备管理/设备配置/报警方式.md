## 获取支持报警方式的能力集
参考:[获取设备能力集接口](https://docs.jftech.com/docs?menusId=ab0ed73834f54368be3e375075e27fb2&siderid=36b2de6403a7424483483189b4736607)
| 属性| 描述 |
| - | - |
| OtherFunction/SupportAlarmVoiceTips| 支持设备报警提示音。更多能力集请参考:[设备能力集](https://docs.jftech.com/docs?menusId=ab0ed73834f54368be3e375075e27fb2&siderid=36b2de6403a7424483483189b4736607)|
|OtherFunction/AlarmOutUsedAsLed|支持设备报警警示灯|

```
DevConfigInfo devConfigInfo = DevConfigInfo.create(new DeviceManager.OnDevManagerListener() {
    @Override

    }
    @Override
    public void onFailed(String devId, int msgId, String s1, int errorId) {

devConfigInfo.setChnId(-1);
devConfigManager.getDevConfig(devConfigInfo);
```

+ 返回的Json字段说明

| 字段| 类型 | 描述 | 取值范围 |
| - | - | - | - |
| Enable | boolean| 动态检测开启| false/true|
| SnapEnable | boolean | 快照使能| false / true|
| SnapShotMask| String| 快照通道号掩码| 16进制的数字字符串|
|RecordEnable|boolean|录像使能|false/true|
|RecordMask|String|录像通道掩码|16进制的数字字符串|
|<span style="color:red"> VoiceEnable </span>|boolean|<span style="color:red"> 语音提示，对应的能力集：OtherFunction/SupportAlarmVoiceTips </span>|false/true|
|<span style="color:red">AlarmOutEnable</span>|boolean|<span style="color:red">报警输出使能(报警警示灯)，对应的能力集：OtherFunction/AlarmOutUsedAsLed</span>|false/true|
|voiceTipInterval|int|设备警铃间隔时间，该值等于0为进一次报警声音提示，大于0为循环报警，间隔时长为设置的该值,单位按秒计算 |大于0的int整数|
|voiceType|int|语音提示类型|根据获取到的警铃类型选择，详情参考设备警铃|
|AlarmOutLatch|int|报警输出延时时间（秒）|10~300|
|AlarmOutMask|String|设备警示灯报警开关 大于0 开启等于0关闭|16进制的数字字符串|
|MessageEnable|boolean|设备报警消息上报服务器|false/true|
|TimeSection|String[][]|事件时间段| "是一个二维数组，每周7天，每天6个时间段。时间段用字符串表示，字符串首的数字1表示该时间段有效，0表示该时间段无效。如果某字符串些事件处理参数中没有这个成员，表示不区分时间段，全天候响应事件。|
|FTPEnable|boolean|Ftp上传使能|false/true|
|LogEnable|boolean|日志使能|false/true|
|MailEnable|boolean|发送邮件，如果有图片，作为附件|false/true|
|MatrixEnable|boolean|联动视频矩阵通道掩码|false/true|
|MatrixMask|String|联动视频矩阵||
|MsgtoNetEnable|boolean|消息上传到网络使能|false/true|
|PtzEnable|boolean|云台联动使能|false/true|
|PtzLink|String[][]|云台联动项|是一个二维数组，每个云台通道对应一个项，每一项包含联动的类型和取值，联动类型有:“None”, “Preset”, “Tour”, “Pattern”|
|Level|int|灵敏度|[1,6]|
|Region|String[]|区域块|是一个数组，区域的每一行使用一个32位整型表示，整型的每一位对应一个小块|
