- 示例代码

```java
 /**
  * 获取省电模式配置
  */
private void getPowerSavingMode() {
    DevConfigInfo devConfigInfo = DevConfigInfo.create(new DeviceManager.OnDevManagerListener<String>() {
        /**
         * 成功回调
         *
         * @param devId         设备类型
         * @param operationType 操作类型
         * @param result        结果
         */
        @Override
        public void onSuccess(String devId, int msgId, String jsonData) {
            // jsonData 结果数据
        }

        /**
         * 失败回调
         *
         * @param devId    设备序列号
         * @param msgId    消息ID【EUIMSG】
         * @param jsonName 参考【JsonConfig】
         * @param errorId  错误码
         */
        @Override
        public void onFailed(String devId, int msgId, String jsonName, int errorId) {
	    //获取失败，通过errorId分析具体原因
        }
    });

    devConfigInfo.setJsonName(JsonConfig.SleepMode);
    devConfigInfo.setChnId(-1);
    devConfigManager.setDevCmd(devConfigInfo);
}

```
- 请求：

```json
{
    "Name": "System.Sleep",
    "SessionID": "0x0000000001"
}
```



- 响应：

```json
{
    "System.Sleep": {
        "DeepSleep": {
          "BatThresholds":30
        },
        "LightSleep": {
            "Mode": 0,
            "Param": [
                {
                    "Value": 0,
                    "Delay": 300
                }
            ]
        }
    },
    "Ret": 100,
    "Name": "System.Sleep",
    "SessionID": "0x0000000001"
}
```


- 字段说明

| 字段名      | 类型 |值范围 | 描述|
| ----------- | ------|---- | --------------------------------- |
| DeepSleep| | | 深度休眠|
| └─BatThresholds| int| 0~100 | 电量百分比 |
| LightSleep| | |浅度休眠|
| └─Mode| int| 0 - 触发模式 - 锁端触发后短时间内进入浅睡模式（该时间可以设置，参考文档下面的Delay），过了这个时间点后设备会进入深水睡状态无法唤醒；<br>  1 - 图传模式 - 锁端被唤醒后上传一张图片成功后立马休眠进入深睡 ；<br>  2 - 监控模式 - 设备处于浅睡状态，支持远程唤醒 |当前模式	|
| └─Param| | | 针对监控模式设置浅睡时间|
| ---└─Value|int|0| 模式值|
| ---└─Delay|int|300 - 浅睡5分钟 <br> 600 - 浅睡10分钟 <br> 900 - 浅睡15分钟 <br> 1800 - 浅睡30分钟| 浅睡时长，单位秒|
