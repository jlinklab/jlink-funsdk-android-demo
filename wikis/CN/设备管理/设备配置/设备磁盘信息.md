- 磁盘信息字段说明

| 名称              | 类型      | 是否必须 | 参考值               | 备注                         |
|-------------------|-----------|----------|:---------------------|------------------------------|
| Name           | string    | 必须     |                      | 名称                         |
| Ret            | int       | 必须     | 100                  |                              |
| SessionID | string | 必须 | 0x00000196 | |
| StorageInfo    | object[]  |          |                      |                              |
|    └─ PlysicalNo | int       |          |                      | 磁盘物理序号                 |
| └─ PartNumber | int | | | 磁盘分区数 |
| └─ Partition | object | | |  |
|    ──└─ DirverType    | int       |      | 0                    | 磁盘类型,读写:0,只读:1,冗余:3,快照:4 |
|    ──└─ TotalSpace    | int       |      | 2048000              | 总大小，M为单位          |
|   ──└─ RemainSpace   | int       |      | 2048000              | 剩余大小，M为单位              |
|     ──└─ IsCurrent     | boolean   |      | false                | 是否为工作盘         |
| ──└─ LogicSerialNo | int       |      | 0                    | 驱动器逻辑序号：磁盘号加分区号               |
|     ──└─ NewEndTime     | string    |      | 0000-00-00 00:00:00 | 新录像文件结束时间           |
|   ──└─ NewStartTime   | string    |      | 0000-00-00 00:00:00 | 新录像文件开始时间           |
|     ──└─ OldEndTime     | string    |      | 0000-00-00 00:00:00 | 上一个录像文件结束时间       |
|   ──└─ OldStartTime   | string    |      | 0000-00-00 00:00:00 | 上一个录像文件开始时间       |
|       ──└─ Status       | int       |      | 0                    | 驱动器错误标志               |
