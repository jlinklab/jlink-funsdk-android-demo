
- 代码示例

```java
 /**
  * 获取支持的语言列表
  */
private void getDevSupportedLanguages () {
    DevConfigInfo devConfigInfo = DevConfigInfo.create(new DeviceManager.OnDevManagerListener<String>() {
        /**
         * 成功回调
         *
         * @param devId         设备类型
         * @param operationType 操作类型
         * @param result        结果
         */
        @Override
        public void onSuccess(String devId, int msgId, String jsonData) {
            // jsonData 结果数据
        }

        /**
         * 失败回调
         *
         * @param devId    设备序列号
         * @param msgId    消息ID【EUIMSG】
         * @param jsonName 参考【JsonConfig】
         * @param errorId  错误码
         */
        @Override
        public void onFailed(String devId, int msgId, String jsonName, int errorId) {
			//获取失败，通过errorId分析具体原因
        }
    });


    devConfigInfo.setCmdId(1360);
    devConfigInfo.setJsonName(JsonConfig.DEVICE__SUPPORT_LANGUAGE);
    devConfigInfo.setChnId(-1);
    devConfigManager.setDevCmd(devConfigInfo);
}
```

- 请求：

```json
{
    "Name": "MultiLanguage",
    "SessionID": "0x00000001e3"
}
```



- 响应：

```json
{
    "MultiLanguage": [
        "English",//英文
        "SimpChinese",//简体中文
        "ChineseEnglish"//繁体中文
    ],
    "Name": "MultiLanguage",
    "Ret": 100,
    "SessionID": "0x000001e3"
}
```


- 字段说明

| 字段名      | 类型 | 描述                              |
| ----------- | ---- | --------------------------------- |
| MultiLanguage | String[]| 语言类型|
