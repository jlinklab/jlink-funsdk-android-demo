## 获取支持移动追踪能力
参考:[获取设备能力集接口](https://docs.jftech.com/docs?menusId=ab0ed73834f54368be3e375075e27fb2&siderid=36b2de6403a7424483483189b4736607)
| 属性                               | 描述                                       |
| -------------------------------- | ---------------------------------------- |
| AlarmFunction/MotionHumanDection | 移动追踪,完整的能力集参考:[设备能力集](https://docs.jftech.com/docs?menusId=ab0ed73834f54368be3e375075e27fb2&siderid=36b2de6403a7424483483189b4736607) |

## 获取配置
- 示例代码

``````java
DevConfigInfo devConfigInfo = DevConfigInfo.create(new DeviceManager.OnDevManagerListener() {
    @Override
    public void onSuccess(String devId, int msgId, Object result) {
        //result是json数据
    }

    @Override
    public void onFailed(String devId, int msgId, String s1, int errorId) {
        //获取失败，通过errorId分析具体原因
    }
});

devConfigInfo.setJsonName(JsonConfig.CFG_DETECT_TRACK);
devConfigInfo.setChnId(-1);
devConfigManager.getDevConfig(devConfigInfo);
``````

- 请求

``````json
{
    "Name": "Detect.DetectTrack",
    "SessionID": "0x0000003f"
}

``````

- 响应

``````json
{
    "Detect.DetectTrack": {
        "Enable": 0,
        "ReturnTime": 300,
        "Sensitivity": 1
    },
    "Name": "Detect.DetectTrack",
    "Ret": 100,
    "SessionID": "0x0000003f"
}

``````

- 返回的Json字段说明

| 字段          | 类型   | 描述                 | 取值范围                                     |
| ----------- | ---- | ------------------ | ---------------------------------------- |
| Enable      | int  | 移动追踪开关             | 0:关闭 1:开启;默认值:0                          |
| ReturnTime  | int  | 移动追踪结束后 返回守望点的等待时间 | 守望时间单位s（秒），支持选择的时间【3,5,10,15,30,60,180,300,600,900,1800】; 默认值:300 |
| Sensitivity | int  | 移动追踪灵敏度            | 0:灵敏度低 1:灵敏度中 2:灵敏度高;默认值:2               |
