### 1. 固件升级检测
- 接口说明

```
void checkDevUpgrade(String devId, DeviceManager.OnDevUpgradeListener listener);
```
- 参数说明

| 参数     | 类型  | 描述     | 取值范围|
| :--------- | -------- |---- |---- |
| devId |String |设备序列号   | 非空|
| listener |OnDevUpgradeListener  |回调| 非空|

- 示例代码

```
DeviceManager.getInstance().checkDevUpgrade(getDevId(), new DeviceManager.OnDevUpgradeListener() {
    /**
     * 检测版本升级回调
     *
     * @param upgradeType 升级类型
     * @param verInfo     升级版本信息
     * @param errorId     错误Id
     */
      @Override
      public void onCheckUpgradeResult(int upgradeType, String verInfo, int errorId) {

      }
      /**
       * 升级回调
       *
       * @param upgradeState 升级状态   //正在下载升级包 ECONFIG.EUPGRADE_STEP_DOWN 正在上传 ECONFIG.EUPGRADE_STEP_UP 正在升级 ECONFIG.EUPGRADE_STEP_UPGRADE 升级完成 ECONFIG.EUPGRADE_STEP_COMPELETE
       * @param progress 根据upgradeState值判断是固件下载进度、固件上传进度和升级进度
       * @param errorId 升级错误
       * @param info    升级过程中带的信息
       */
      @Override
      public void onUpgradeProgress(int state, int progress, int errorId, String info) {

      }
  });
```

### 2. 开始设备固件升级
- 接口说明

```
 void startDevUpgrade(String devId, int upgradeType, DeviceManager.OnDevUpgradeListener listener);
```
- 参数说明

| 参数  | 类型     | 描述    | 取值范围|
| :--------- | -------- |----- |----- |
| devId |String |设备序列号   | 非空|
| upgradeType| int |升级类型 |  |
| listener|OnDevUpgradeListener |回调| 非空|

- upgradeType 升级类型

| 参数       | 描述     |
| :--------- | -------- |
| UPGRADE_TYPE_NONE |没有可升级的   |
| UPGRADE_TYPE_CLOUD|云升级 |
| UPGRADE_TYPE_FILE_DOWNLOAD|固件下载后升级|
| UPGRADE_TYPE_LOCAL_FILE|选择本地固件升级|

- 示例代码

```
DeviceManager.getInstance().startDevUpgrade(getDevId(),upgradeType, new DeviceManager.OnDevUpgradeListener() {
    /**
     * 检测版本升级回调
     *
     * @param upgradeType 升级类型
     * @param verInfo     升级版本信息
     * @param errorId     错误Id
     */
      @Override
      public void onCheckUpgradeResult(int upgradeType, String verInfo, int errorId) {

      }
      /**
       * 升级回调
       *
       * @param upgradeState 升级状态   //正在下载升级包 ECONFIG.EUPGRADE_STEP_DOWN 正在上传 ECONFIG.EUPGRADE_STEP_UP 正在升级 ECONFIG.EUPGRADE_STEP_UPGRADE 升级完成 ECONFIG.EUPGRADE_STEP_COMPELETE
       * @param progress 根据upgradeState值判断是固件下载进度、固件上传进度和升级进度
       * @param errorId 升级错误
       * @param info    升级过程中带的信息
       */
      @Override
      public void onUpgradeProgress(int state, int progress, int errorId, String info) {

      }
  });
```

### 4. 升级回调函数
```
  public interface OnDevUpgradeListener {
        /**
         * 检测版本升级回调
         *
         * @param upgradeType 升级类型
         * @param verInfo     升级版本信息
         * @param errorId     错误Id

        /**
         * 升级回调
         *
         * @param upgradeState 升级状态   //正在下载升级包 ECONFIG.EUPGRADE_STEP_DOWN 正在上传 ECONFIG.EUPGRADE_STEP_UP 正在升级 ECONFIG.EUPGRADE_STEP_UPGRADE 升级完成 ECONFIG.EUPGRADE_STEP_COMPELETE
         * @param progress 根据upgradeState值判断是固件下载进度、固件上传进度和升级进度
         * @param errorId 升级错误
         * @param info    升级过程中带的信息
         */
        void onUpgradeProgress(int upgradeState, int progress, int errorId, String info);
    }
