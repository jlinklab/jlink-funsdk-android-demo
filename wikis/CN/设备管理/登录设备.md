### 1. 获取设备状态（登录设备之前先获取设备状态）
#### 1.1. 获取单个设备状态：

- 接口说明

```
void updateDevStateFromServer(BaseAccountManager.OnDevStateListener listener, String devId);
```
- 参数说明


| 参数     | 类型  | 描述     |取值范围 |
| :------|--- | -------- | --- |
| listener |`OnDevStateListener `| 回调   | 非空|
| devId| `String`| 设备序列号 | 非空|


- 示例代码

```
XMAccountManager.getInstance().updateAllDevStateFromServer( AccountManager.getInstance().getDevList(), new BaseAccountManager.OnDevStateListener() {
    @Override
    public void onUpdateDevState(String devId) {
        //单个设备状态回调
        XMDevInfo xmDevInfo = DevDataCenter.getInstance().getDevInfo(devId);
        //获取设备状态
        xmDevInfo.getDevState()
    }

    @Override
    public void onUpdateCompleted() {
        //所有设备状态获取结束回调
    }
});
```


#### 1.2. 获取所有设备状态:
- 接口说明

```
void updateAllDevStateFromServer(List<String> devIdList, BaseAccountManager.OnDevStateListener listener);
```
- 参数说明

| 参数     | 类型  | 描述     |取值范围 |
| :------|--- | -------- | --- |
| devIdList| `List<String>` | 设备列表   | 非空|
| listener |`OnDevStateListener` | 回调   | 非空|

- 示例代码

```
XMAccountManager.getInstance().updateAllDevStateFromServer( XMAccountManager.getInstance().getDevList(), new BaseAccountManager.OnDevStateListener() {
    @Override
    public void onUpdateDevState(String devId) {
        //单个设备状态回调
    }

    @Override
    public void onUpdateCompleted() {
        //所有设备状态获取结束回调
    }
});
```

### 2. 登录设备
#### 2.1. 普通设备登录:
- 接口说明

```
boolean loginDev(String devId, DeviceManager.OnDevManagerListener listener);
```
- 参数说明

| 参数     | 类型  | 描述     |取值范围 |
| :------|--- | -------- | --- |
| devId| `String` | 设备序列号   | 非空|
| listener |`OnDevManagerListener ` | 回调   | 非空|

- 示例代码

```
DeviceManager.getInstance().loginDev(devId, new DeviceManager.OnDevManagerListener() {
    /**
     * 成功回调
     *
     * @param devId         设备类型
     * @param operationType 操作类型
     * @param result        结果
     */
    @Override
    public void onSuccess(String devId, int operationType, Object o) {

    }

     /**
     * 失败回调
     *
     * @param devId    设备序列号
     * @param msgId    消息ID
     * @param jsonName
     * @param errorId  错误码
     */
    @Override
    public void onFailed(String devId, int msgId, String jsonName, int i1) {

    }
});

```
